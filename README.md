## Brain-games

Set of five console games based on popular mobile brain training apps.

## System requires

- node.js 12 or higher
- make

## Install

```bash
make install
```

## Run games

```bash
make brain-even
make brain-calc
make brain-gcd
make brain-progression
make brain-prime
```
